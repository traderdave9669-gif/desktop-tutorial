# BB4L BTC Dashboard

This repo contains a static dashboard page for the BB4L BTC edge tool.

## Cloudflare deployment

1. Upload the files in this folder to a Cloudflare Pages or static hosting target.
2. In Cloudflare Zero Trust, create an Access application for the dashboard hostname.
3. Require login for approved users, or allowlist specific emails/IPs.
4. Keep the dashboard private so only authorized viewers can access it.
